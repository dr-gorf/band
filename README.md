# The Website
[SUPPLE CONGRESS](https://supple-congress.github.io "SUPPLE CONGRESS")

### Build
```sh
# install dependencies
$ npm install

# runs webpack
$ npm run build
```

### Run locally
```sh
# starts a server on localhost:8080
$ npm start
```

### Deploy
```sh
# runs build and pushes contents of /dist to master
$ npm run deploy
```
